# 项目结构

## 目录结构

```
Gg/
├── CMakeLists.txt           # 根级构建配置
├── CMakePresets.json        # 构建预设
├── cmake/config.cmake       # 全局编译配置
├── include/Gg/              # 公共头文件
│   ├── AppTypes.h          # 类型定义
│   ├── AppController.h     # 控制器
│   ├── Gg.h                # 主窗口
│   ├── MetricsEngine.h    # 核心引擎
│   ├── WorkspaceRepository.h # 数据仓库
│   ├── WinDeviceInfo.h    # Windows设备信息
│   ├── TrajectoryWidget.h # 轨迹组件
│   ├── SparklineWidget.h  # 图表组件
│   └── LambdaThread.h     # 线程工具类
├── src/                     # 源文件
│   ├── main.cpp
│   ├── Gg.cpp
│   ├── AppController.cpp
│   ├── MetricsEngine.cpp
│   ├── WorkspaceRepository.cpp
│   ├── WinDeviceInfo.cpp
│   ├── TrajectoryWidget.cpp
│   ├── SparklineWidget.cpp
│   └── LambdaThread.cpp
├── lib/                     # 模块化库
│   ├── types/              # GgTypes (接口库)
│   ├── common/             # GgCommon (工具类)
│   ├── data/               # GgData (数据层)
│   ├── core/               # GgCore (核心业务)
│   ├── device/             # GgDevice (设备层)
│   └── ui/                 # GgUi (UI组件)
└── docs/                    # 文档
```

## 模块说明

| 模块 | 库名 | 说明 |
|------|------|------|
| types | GgTypes | AppTypes.h，类型定义和JSON序列化工具 |
| common | GgCommon | LambdaThread，异步线程工具 |
| data | GgData | WorkspaceRepository，数据持久化 |
| core | GgCore | MetricsEngine，指标计算核心 |
| device | GgDevice | WinDeviceInfo，Windows设备信息 |
| ui | GgUi | TrajectoryWidget, SparklineWidget |

## 依赖关系

```
Gg (可执行文件)
├── GgUi
│   └── GgTypes
├── GgDevice
│   └── GgTypes
├── GgCore
│   └── GgTypes
├── GgData
│   └── GgTypes
├── GgCommon
│   └── GgTypes
└── GgTypes (仅头文件)
```

## 构建命令

```bash
# Debug
cmake -B build -DCMAKE_BUILD_TYPE=Debug
cmake --build build --config Debug

# Release
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
```

输出: `build/bin/Debug/Gg.exe` 或 `build/bin/Release/Gg.exe`
