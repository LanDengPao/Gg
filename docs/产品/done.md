# 已完成

## 当前可用功能

- 已完成 Windows + Qt Widgets 桌面端基础工程与 CMake 构建。
- 已接入 Windows Raw Input，可实时采集鼠标移动、滚轮、左/右/中键事件。
- 已实现首页 / 测试 / 历史 / 对比 / 设备 5 个主页面。
- 已实现测试模式切换：轮询率测试、轨迹/稳定性测试。
- 已支持手动开始测试、停止测试，并生成一次会话记录。
- 已实现实时指标计算：当前 Hz、窗口平均 Hz、最大值、最小值、标准差、稳定度、速度、抖动、点击计数。
- 已实现轮询率曲线与鼠标轨迹图展示。
- 已实现会话落盘保存，包含 session 元数据、summary 汇总、samples 原始采样。
- 已实现历史记录列表展示。
- 已实现多次测试结果对比表。
- 已支持导出原始采样数据，格式包括 CSV 和 JSON。
- 已实现设备基础信息识别与展示，包含名称、VID、PID、连接类型等。
- 已实现中英文界面切换。

## 本轮已修复

- 已修复轨迹图方向显示问题，轨迹绘制不再上下反转。
- 已修正平均 Hz 的计算口径，改为基于采样间隔的等效平均频率。
