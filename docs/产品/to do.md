# 产品提需求

## 待办

### 高优先级
- 明确 Hz 统计口径并统一到全部指标展示与历史汇总中（当前已修正平均值，仍可继续统一 1s/5s/30s、min/max、稳定度口径）。
- 补充轨迹/稳定性测试的结果表达，区分“轮询率测试”和“轨迹测试”各自更关注的核心指标。
- 优化稳定度评分算法，使其与目标轮询率（如 125/500/1000Hz）更贴近。
- 增加原始输入事件计数、实时坐标、Delta X / Delta Y 展示。
- 完善设备信息识别，补充厂商、产品名、固件版本、电量等可获取字段。

### 中优先级
- 增加轨迹图、轮询曲线在历史详情中的回看能力，而不只是表格汇总。
- 增加历史记录详情页或弹窗，展示单次测试完整 summary。
- 优化多鼠标切换场景下的当前活动设备判定与显示。
- 补充导出内容，支持 summary 单独导出或附带统计结果导出。
- 完善异常状态提示，如无设备、文件写入失败、设备切换等场景。

### 低优先级 / 后续扩展
- 增加点击时间线、滚轮时间线、速度变化曲线。
- 增加对比页更直观的图表化展示。
- 支持图表截图导出或报告导出。
- 预留 HID / 厂商 SDK 扩展能力，提升专业设备信息获取深度。
- 评估点击延迟、DPI / CPI 估算、传感器异常点等高级测试能力。
